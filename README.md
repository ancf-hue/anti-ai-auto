📖 О проекте
Анти ИИ Автомобиль — открытое веб-приложение для предварительной оценки состояния силового агрегата по виброакустическим характеристикам. Приложение использует встроенный акселерометр мобильного устройства, анализирует векторную сумму колебаний по трём осям (X, Y, Z) и сравнивает текущий замер с сохранённым базовым профилем.
🔹 Без облаков, без нейросетей, без скрытых алгоритмов. Только детерминированная математика, прозрачная логика и полный контроль данных у пользователя.
⚙️ Принцип работы
Сбор данных: Акселерометр фиксирует ускорения по осям с частотой ~50–60 Гц.
Векторный анализ: Вычисляется модуль вибрации: V = √(x² + y² + z²).
Фильтрация: Применяется экспоненциальное сглаживание и компенсация фонового шума.
Профилирование: 30-секундный замер разбивается на 20 сегментов, формируется эталонный профиль.
Сравнение: Текущий профиль сопоставляется с сохранённым эталоном, рассчитывается процент отклонения.

Быстрый запуск
Создайте файл index.html и вставьте код проекта.
Добавьте в ту же директорию:
manifest.json (для PWA)
sw.js (Service Worker)
frame.jpg, cube.mp4, logo.jpeg
Папку sounds/ с аудиофайлами (ready.mp3, sensor_ok.mp3 и др.)
Запустите через локальный сервер с HTTPS или localhost (доступ к датчикам требует защищённого контекста): 📱 Руководство пользователя
Подготовка: Прогрейте двигатель до рабочей температуры. Установите автомобиль на ровную площадку.
Проверка датчика: Нажмите 🔍 ПРОВЕРИТЬ ДАТЧИК. Убедитесь, что статус стал зелёным.
Фиксация эталона:
Положите телефон горизонтально на панель приборов.
Нажмите 📥 СОХРАНИТЬ ЭТАЛОН.
Покиньте салон, закройте двери. Через 30 сек данные сохранятся.
Сравнительный анализ:
Повторите установку телефона в то же положение.
Нажмите 📊 СРАВНИТЬ.
Получите процент отклонения и рекомендации.
Мониторинг: В разделе 📜 ИСТОРИЯ используйте 📊 СРАВНИТЬ ПО ДАТАМ для отслеживания динамики за месяцы/годы.
